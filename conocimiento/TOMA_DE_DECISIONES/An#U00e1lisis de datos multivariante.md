## 📝 Descripción

Asignatura cuantitativa enfocada en el análisis simultáneo de múltiples variables para identificar patrones, relaciones y estructuras en conjuntos de datos complejos. Integra técnicas como análisis factorial, de conglomerados, discriminante y de componentes principales.

---

## 🎯 Aprendizajes clave

- Aplicar técnicas multivariantes para el análisis de datos complejos
- Interpretar resultados en contextos organizacionales
- Identificar patrones y relaciones en datos multidimensionales
- Tomar decisiones basadas en análisis avanzados de datos

---

## 💼 Valor profesional

Permite desarrollar capacidades avanzadas de análisis de datos para la toma de decisiones estratégicas, con alta aplicabilidad en investigación de mercados, análisis financiero y gestión organizacional.
