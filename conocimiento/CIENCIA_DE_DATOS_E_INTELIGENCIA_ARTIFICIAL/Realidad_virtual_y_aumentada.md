## 📝 Descripción

Esta materia te enseña a crear experiencias digitales usando realidad virtual, aumentada y mixta. Aprendes a desarrollar soluciones creativas para representar y manejar datos con herramientas tecnológicas. Trabajas con programación y entornos digitales para resolver problemas de forma innovadora. Se recomienda contar con bases en análisis de datos y programación

---

## 🎯 Aprendizajes clave

- Al finalizar la materia, serás capaz de:
- Distingue entre realidad virtual, aumentada y mixta y sus usos
- Analiza tecnologías actuales para crear soluciones digitales interactivas
- Diseña soluciones innovadoras con VR/AR para problemas reales
- Utiliza plataformas para desarrollar aplicaciones VR/AR/MR

## 💼 Valor profesional
